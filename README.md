## About our project
- Treat Your-shelf is a personalized AI pantry assistant that bridges the gap between food insecurity and goal-oriented nutrition.
- Converts the dietary restrictions, allergies, and goals into a health profile
- Uses Gemini 2.5 Flash-Lite to analyzed the picture for the ingredients.
- Then VectorAI DB to search for and finds recipes with ingredients from the analyzed picture/ upload.

### Technologies used:
`Actian VectorAI DB` `Gemini API`  `React` `Tailwind`

- More agent tooling for more preferential search.
